# open-project-templates

Welcome to **open-project-templates** — a collection of ready-to-use project templates designed to help you bootstrap new projects quickly and consistently.

## About

This organisation provides opinionated, production-ready starter templates for a variety of technologies and workflows. Each template follows community best practices and is designed to be cloned and customised with minimal effort.

## Available Templates

| Technology | Description |
|---|---|
| 🐳 **Docker** | Dockerfile and Docker Compose templates for containerising applications |
| ⚙️ **Ansible** | Playbook and role templates for infrastructure automation |
| 🐹 **Go** | Project scaffolding for Go applications and libraries |
| …and more | Additional templates covering CI/CD pipelines, Kubernetes, Terraform, and other tools |

## Getting Started

1. Browse the repositories in this organisation to find a template that suits your needs.
2. Click **Use this template** (or clone the repository) to create your own copy.
3. Follow the `README.md` inside each template for setup and customisation instructions.

Contributions are welcome! If you have a template you'd like to share, please open an issue or pull request in the relevant repository.
